# OmniHub Project

OmniHub is an all-in-one smart web platform that connects users to various services, including entertainment, shopping, AI tools, games, study resources, and more. This project aims to provide a seamless experience for users to access these services from a single platform.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd omnihub
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

- Open `src/pages/index.html` in your browser to view the homepage.
- Use the search bar to find services quickly.
- Explore different categories to discover various tools and resources.
